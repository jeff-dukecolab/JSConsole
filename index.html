The Console

What is it?

The console is an object that contains a wonderful assortment of methods. Modern browsers can display console information in the `web console`.

> typeof(console);
<- "object"

How do I use it?

  • chrome - right click anywhere on the page and select inspect
  • firefox - right click anywhere on the page and select inspect element
  • edge - right click anywhere on the page and select inspect

Try it out!

Open up your console and smash these buttons!
